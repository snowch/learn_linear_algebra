Definition: Vector

We begin with the basics of vectors. A vector in mathematics is an object that has both magnitude and direction. In the context of linear algebra, vectors are essential building blocks used in various operations and applications.

2D Vector#

First start with 2D to keep concepts simple for understanding basic functionality.

Linear Combination = both Scalar multiplication and Vector Addition

c1 = 2

v2 = vector([1,0])

lin_combination = c1*v1 + c2*v2

plots = \
    v1.plot(color='blue') + \
    (c1*v1).plot(color='lightblue', linestyle=':') + \
    v2.plot(color='blue') + \
    lin_combination.plot(color='red')

show(plots, figsize=4)
